<h1>📌 About This Project</h1>

This project demonstrates how JavaScript Events work and how they make websites interactive.

I implemented different event types such as:

🖱 Click Events

⌨ Keyboard Events

📝 Form Events

🎯 Mouse Events

The goal was to understand how JavaScript listens and responds to user actions.
